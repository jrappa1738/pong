# pong
machine vision project for autonomous Atari pong playing - Oregon Institute of Technology - EE407 Winter 2018

To use: 

git clone https://github.com/jrappa1738/pong\n
cd ../pong\n
mkdir build\n
cd build\n
cmake ..\n
make\n
./final_proj\n


Steps to tune:
Adjust input quadrilateral for perspective transform (TrackbarsCoordinates)
Adjust bottom paddle color threshold (TrackbarBotPaddle)
